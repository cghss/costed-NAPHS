# Costed National Action Plans for Public Health (NAPHS)

This repository contains publicly available data on costed National Action Plans for Health Security (NAPHS) and associated analyses. It is currently a work-in-progress. Please stay tuned!

**Data:** Contains both raw and clean datasets. You can access the final clean dataset in the clean/directory, including a data dictionary.

**Code:** Contains all code used to import, clean, export, and analyze the datasets.

**Documentation:** Contains assorted documentation, including the guidelines for data extraction from the original costed NAPHS PDF files.

**Results:** Contains results of analyses, including both summary tables and figures.
