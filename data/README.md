# Costed National Action Plans for Public Health (NAPHS)
## Data directory

This repository contains raw (manually extracted from PDFs) and cleaned data from publicly available costed National Action Plans for Health Security (NAPHS).

**Raw/:** The raw/ subdirectory contains raw data, as originally coded by the research team and/or extracted from publicly available PDFs. For more information on data extraction guidelines, please see the **documentation/** directory.

**Clean/:** The clean/ subdirectory contains clean data generated based on the files included in the raw/ subdirectory and processed by the script code/1_data_cleaning.R. As part of data cleaning, separate tabs of an Excel file are merged together and the names of core capacities are standardized across JEE versions.
