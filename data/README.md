# Costed National Action Plans for Public Health (NAPHS)
## Data directory

This repository contains clean data from publicly available costed National Action Plans for Health Security (NAPHS). The **raw/** subdirectory contains raw data that are processed using the script code/1_data_cleaning.R. The **clean/** subdirectory contains the final, clean analytical datasets used for analysis in code/2_analysis.R and code/3_visualizations.R. Each subdirectory contains its own data dictionary documenting the datasets contained within it.

For more information on data extraction guidelines, please see the **documentation/** directory.
